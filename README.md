# Dynamic Cloud Website 🚀

A dynamic web application built using Node.js and Express, deployed on Render.

## 🔧 Tech Used
- Node.js
- Express.js
- Render (Cloud Deployment)
- GitHub (Version Control)

## 🌐 Live Demo
https://dynamic-cloud-website.onrender.com

## 📌 Features
- Dynamic form submission
- Professional UI design
- Submit button animation
- Cloud deployment with auto-deploy from GitHub
