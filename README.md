# 🚀 Dynamic Cloud + AI Portfolio

A fully dynamic cloud-deployed web application built using Node.js and Express, integrated with a small AI/ML (NLP Sentiment Analysis) feature and hosted on Render.

This project demonstrates backend development, cloud deployment, and AI integration in a single application.

## 📌 Features

### 🌍 Cloud Deployment
- Hosted on Render
- Connected to GitHub
- Manual / Auto deploy support

### ⚡ Dynamic Backend Rendering
- Server generates HTML dynamically
- Real-time response on every refresh
- Server Time display
- Server Uptime tracking
- Visitor Counter (session-based)
- Random Motivational Quote
- Random Tech Tip

### 🤖 AI/ML – NLP Sentiment Analyzer
- Accepts user input text
- Uses sentiment analysis library
- Classifies text as:
  - Positive 😊
  - Neutral 🙂
  - Negative 😟
- Displays sentiment score

This demonstrates integration of AI/ML into a backend cloud application.

### 🧠 Additional Functionalities
- Dynamic Greeting Generator
- Submit button loading animation
- Clean responsive UI
- Server-side input sanitization
